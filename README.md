# Factorio crafting graph
A d3.js force-graph of the crafting tree in factorio, to find the perfect factory layout. 

- Nodes point from ingredient to user.
- Root items (uncraftables) are highlighted in green.
- Leaf items (non-ingredients) are highlighted in orange.
- Each node is a link to the factorio wiki page, double click for easy access to more details about each item.
